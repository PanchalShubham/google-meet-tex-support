Google Meet Math Support

Introduction

A Google Chrome extension to parse markdown and tex-equations in Google Meet Chat Window!
The following are the dependencies:

  • markdown-it-texmath is used for parsing markdown and tex-math
  • markdown-it is used for parsing markdown!
  • KaTex is used for rendering Tex math in HTML!
  • markdown-it-emoji is used for parsing emoji!
  • highlight.js is used for syntax highlighting!

The following are important things to note:

  • This extension do not send data to anywhere and therefore it can only parses the messages being receieved in the chat window!
  • The parsed messages are visible to you only in parsed format and others will see the plain text unless they have installed the same extension.
  • To allow the parsing you need to click on the extension everytime you attend a meet because we don't want unnecessary message parsing overhead unless required.
